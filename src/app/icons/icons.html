Icons

Material Symbols Outlined-Icons

Die IconComponent bietet eine einfache Möglichkeit, Material Symbols Outlined-Icons in deiner Angular-App zu verwenden. Sie kapselt das Icon-Styling und stellt eine saubere, wiederverwendbare Schnittstelle bereit, bei der der Icon-Name über Content Projection eingebettet wird.

Die Komponente lädt die Schriftart lokal über ein eingebettetes @font-face und rendert das Icon innerhalb eines zentrierten <span>-Elements.

Einsatzgebiet

Diese Komponente eignet sich für Projekte, in denen:

  • einheitliche Icon-Darstellung benötigt wird
  • Icons zentral gestylt und verwaltet werden sollen
  • Material Symbols Outlined verwendet werden
  • Icons per Content Projection flexibel gesetzt werden sollen

Verwendung

Ein Icon wird als Textinhalt innerhalb des <icon>-Tags übergeben:

Eine Lister mit allen Icons von Google ist hier zu finden

Die benötigte Font dazu ist hier zu finden

    <icon id="home-icon">home</icon>
    <icon id="close-btn">close</icon>
    <icon>menu</icon>

Styling

wichtige Hinweise

  • Die Schriftart wird lokal eingebunden, daher muss der Pfad in deiner Projektstruktur stimmen.
  • Flex-Layout zentriert das Icon sauber innerhalb des Spans.
  • Die Komponente rendert automatisch immer die definierte Icon-Schriftart.

Component Logic

wichtige Hinweise

  • @Input() id: Optionales Attribut zur individuellen Identifikation.
  • Die Komponente benötigt keine weiteren Abhängigkeiten.

Vorteile

  • leichte Wiederverwendbarkeit
  • einfache Integration in jede Angular-Struktur
  • lokaler Font-Load → unabhängig von CDNs
  • klare Trennung von Template, Style und Logik
  • intuitive Nutzung dank Content Projection
