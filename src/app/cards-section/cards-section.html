Cards

Wiederverwendbare Card-Komponente

Die Cards-Komponente stellt einen flexiblen Container mit moderner Optik bereit. Sie bietet eine einheitliche Card-Gestaltung mit Schatten, Rahmen und anpassbaren CSS-Variablen. Durch die Verwendung von ng-content kann beliebiger Inhalt eingefügt werden, was die Komponente extrem vielseitig macht.

Funktionen

Die Cards-Komponente bietet folgende Features:

  • Flexibles Content-Projection-System über ng-content
  • Anpassbare Darstellung über CSS-Variablen
  • Moderne Card-Optik mit Schatten und Rahmen
  • Responsive Layout mit Flexbox
  • Zentrierte Inhaltsausrichtung

CSS-Variablen

Die Komponente stellt drei CSS-Variablen zur Verfügung, die das Aussehen anpassen:

  • --background: Hintergrundfarbe der Card (Standard: rgb(255, 255, 255))
  • --shadow-color: Farbe des Schattens (Standard: rgba(0, 0, 0, 0.12))
  • --border-color: Rahmenfarbe (Standard: rgb(227, 227, 227))

Styling-Eigenschaften

Die Card verfügt über folgende visuelle Merkmale:

  • Padding: 1rem vertikal, 2rem horizontal
  • Border-Radius: 0.5rem
  • Box-Shadow: 1px 2px 4px
  • 1px solider Rahmen
  • Vollständige Breite (100%)

Verwendung

Grundlegende Verwendung mit beliebigem Content:

<cards>
    <h2>Card Titel</h2>
    <p>Beliebiger Inhalt kann hier eingefügt werden.</p>
</cards>

Beispiel mit Styling-Anpassung

Die CSS-Variablen können überschrieben werden, um das Aussehen anzupassen:

<cards style="--background: rgb(240, 248, 255); --border-color: rgb(100, 150, 200);">
    <h3>Benutzerdefinierte Card</h3>
    <p>Diese Card hat eine angepasste Hintergrund- und Rahmenfarbe.</p>
</cards>

Beispiel mit komplexem Inhalt

<cards>
    <div>
        <h2>Produktname</h2>
        <p>Produktbeschreibung mit Details...</p>
        <button>Mehr erfahren</button>
    </div>
</cards>

Tipp: Die Cards-Komponente funktioniert besonders gut in Kombination mit der GirdCards-Komponente für responsive Layouts.

GirdCards

Responsive Grid-Layout für Cards

Die GirdCards-Komponente stellt ein automatisches Grid-Layout bereit, das Cards optimal anordnet. Durch CSS Grid mit auto-fit und minmax passt sich das Layout automatisch an die verfügbare Breite an. Perfekt für Dashboard-Layouts, Produkt-Übersichten oder Galerie-Ansichten.

Funktionen

Die GirdCards-Komponente bietet folgende Features:

  • Automatisches responsive Grid-Layout
  • Dynamische Spaltenanzahl basierend auf verfügbarer Breite
  • Minimale Spaltenbreite von 300px
  • Gleichmäßige Abstände zwischen Cards (1rem Gap)
  • Content Projection für beliebige Child-Elemente

Grid-Eigenschaften

Das Grid verwendet folgende CSS-Grid-Konfiguration:

  • display: grid: Aktiviert CSS Grid Layout
  • grid-template-columns: repeat(auto-fit, minmax(300px, 1fr)): Responsive Spalten
  • gap: 1rem: Einheitlicher Abstand zwischen Cards
  • justify-content: center: Zentrierte Ausrichtung

Hinweis: Die Mindestbreite von 300px stellt sicher, dass Cards auch auf mobilen Geräten gut lesbar bleiben.

Responsive Verhalten

Das Grid passt sich automatisch an verschiedene Bildschirmgrößen an:

  • Desktop: Mehrere Spalten nebeneinander (abhängig von der Breite)
  • Tablet: 2-3 Spalten
  • Mobile: 1 Spalte (wenn Breite < 300px + Spacing)

Verwendung

Grundlegende Verwendung mit mehreren Cards:

<app-gird-cards>
    <cards>Card 1 Inhalt</cards>
    <cards>Card 2 Inhalt</cards>
    <cards>Card 3 Inhalt</cards>
</app-gird-cards>

Beispiel mit vollständigem Dashboard-Layout

<app-gird-cards>
    <cards>
        <h3>Benutzer</h3>
        <p>Total: 1.234</p>
    </cards>
    
    <cards>
        <h3>Umsatz</h3>
        <p>€45.678</p>
    </cards>
    
    <cards>
        <h3>Bestellungen</h3>
        <p>567 heute</p>
    </cards>
    
    <cards>
        <h3>Konversionsrate</h3>
        <p>3.2%</p>
    </cards>
</app-gird-cards>

Beispiel mit verschachtelten Inhalten

<app-gird-cards>
    <cards>
        <img src="product1.jpg" alt="Produkt 1">
        <h4>Produkt 1</h4>
        <p>Beschreibung des Produkts</p>
        <button>In den Warenkorb</button>
    </cards>
    
    <cards>
        <img src="product2.jpg" alt="Produkt 2">
        <h4>Produkt 2</h4>
        <p>Beschreibung des Produkts</p>
        <button>In den Warenkorb</button>
    </cards>
</app-gird-cards>

Verwendung ohne Cards-Komponente

Das Grid kann auch mit anderen Elementen verwendet werden:

<app-gird-cards>
    <div class="custom-card">Element 1</div>
    <div class="custom-card">Element 2</div>
    <div class="custom-card">Element 3</div>
</app-gird-cards>

Best Practice: Kombiniere GirdCards mit der Cards-Komponente für ein konsistentes und professionelles Layout-System.
